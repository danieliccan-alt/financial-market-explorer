# Financial Market Explorer
This is a Python data analysis project which aims to read stock price data from a CSV,
calculate its returns, and ranks the stocks from highest to lowest return.

## Technologies used
For this project I used Python, Pandas and Jupyter Notebook.

## Motivation
I was inspired to make this project by my interest in how finance can be integrated
within technology as well as improving my Python skills.

## Features
- Reads stock data from a CSV file
- Calculates percentage return for each stock
- Ranks the stocks from highest to lowest return

## How to run?
1. Clone the repository
2. Create a virtual environment: 'python -m venv venv'
3. Activate it: 'source venv/bin/activate'
4. Install its dependencies: 'pip install pandas'
5. Open the notebook in the 'notebooks\' folder and run all cells.
